## Netflix 南美账号共享试用

Netflix 正在智利、哥斯达黎加和秘鲁试用新的帐户共享政策。Netflix 的计划是将共享货币化，这样人们就必须为添加子帐户或与其他家庭的人共享密码支付一点额外费用。

最初，智利、哥斯达黎加和秘鲁的用户对试用表示不满，以至于订阅人数下降，Netflix 股价和股票报价暴跌。但由于拉丁美洲的 Netflix 订户数量最少，Netflix 似乎并不太担心。

试用的新功能包括可以以比创建新帐户的成本低得多的价格添加两个额外用户。

由于对 Netflix 认为的家庭的批评和困惑，Netflix 似乎必须在推出之前修改其密码共享方案。Netflix 声称将在 2022 年底开始在全球范围内推出这一方案。但在它成为全球推出的解决方案之前，Netflix 需要让子账户系统更加清晰。
